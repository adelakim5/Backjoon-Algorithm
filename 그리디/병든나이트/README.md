## 병든 나이트(1783)
> 그리디

### 풀이
1. 칸의 높이와 너비에 따라 이동 가능한 칸의 최대개수를 구한다. 
    - n = 1 : 시작점 1칸을 출력한다. 
    - n = 2 : Math.min(4, (m+1)/2)칸을 출력한다. 
    - n >= 3
      - m < 7 : Math.min(4, m) 을 출력한다. 
      - m >= 7 : m-7+5을 출력한다.

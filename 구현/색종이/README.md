## 색종이(2563)
> 구현

### 풀이 
1. 흰도화지에 색종이를 덧댈때마다 방문표시를 해준다. 
   -  도화지 전체를 이차원배열로 만들고, false로 채운다. 
   - 색종이의 위치와 크기만큼 도화지를 true로 바꾼다. 
   - 만약 이미 true라면? 건너뛴다.
   - 그렇지 않다면? 해당 부분을 true로 바꾸고, 그만큼 카운트를 세어준다(=> 합쳐진 카운트가 해당 색종이를 덧대었을때 나온 면적)

2. 각 색종이마다 세어준 카운트를 모두 합산하여 출력한다.

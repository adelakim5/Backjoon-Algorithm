## 비밀번호 찾기(17219)

> 해시를 사용한 집합과 맵

### 풀이

1. 사이트를 저장할 오브젝트를 하나 만든다.
   - Map으로 해도 상관없다.
2. 사이트와 비밀번호를 오브젝트에 저장한다.
   - key값을 사이트로 저장한다.
   - value를 비밀번호로 저장한다.
3. 오브젝트 내에서 key에 맞는 value를 찾는다.
   - key값이 사이트이고 value를 비밀번호로 저장했기 때문에 사이트로 비밀번호를 바로 찾을 수 있다.
   - 찾은 비밀번호를 줄바꿈하여 문자열로 엮는다.
4. 엮은 문자열을 출력한다.
